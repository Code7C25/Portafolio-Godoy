Programa para Pastelería Albura

Sistema de gestión integral desarrollado específicamente para una pastelería artesanal.

Funcionalidades principales

  • Control de inventario: Seguimiento en tiempo real de materias primas
  • Calendario de producción: Agenda actualizable para anotar pedidos
  • Base de recetas: Biblioteca digital con fórmulas y procedimientos
  • Calculadora de costos: Automatización de precios basada en ingredientes

Tecnologías usadas

Python Tkinter SQLite Word DB Browser

Proceso de desarrollo

El proyecto incluyó:

  1. Reuniones con la clienta para determinar requerimientos
  2. Diseño de la interfaz gráfica adaptada a sus necesidades
  3. Implementación de la base de datos relacional
  4. Capacitación para el uso del sistema
